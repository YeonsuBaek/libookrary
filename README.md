# 라이북러리

가상의 책꽂이에 읽은 책을 보관하세요.

## 참여하기

다음 버전이 필요합니다: Node.js 16.20.2, Npm 8.19.2

```
git clone https://github.com/YeonsuBaek/libookrary.git
npm install
npm run dev
```
